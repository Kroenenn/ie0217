bool esNumeroPrimo ( unsigned int  numero)

Determina si un número es primo.

Esta función comprueba si un número dado es primo, es decir, solo divisible por 1 y por sí mismo.

Parameters
numeroEl número que se quiere verificar si es primo.
Returns
true si el número es primo, false en caso contrario.

Encuentra el siguiente número primo después de un número dado.

Esta función busca el menor primo que es mayor al número proporcionado. Utiliza una iteración simple para encontrar el siguiente primo.

Parameters
numeroEl número base para buscar el siguiente primo.
Returns
El siguiente número primo mayor que el número dado.
