Detailed Description

Archivo principal del programa Validador de correos electrónicos.

Este archivo contiene la función main que permite al usuario ingresar un correo electrónico y determinar si es válido o no. El usuario puede ingresar un correo electrónico y especificar si el correo tiene una extensión compuesta. El programa utiliza la clase ValidadorEmail para validar el correo electrónico ingresado.
